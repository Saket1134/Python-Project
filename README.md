# First Python Project on Snake Water Gun Game
A classic implementation of the Snake, Water, and Gun (SWG) game in Python. Play against the computer and test your luck and strategy!
Single-player gameplay: Play against an AI opponent.

Multiple rounds: The game is played for a predefined number of rounds (or best-of-N).

Score tracking: Keeps track of player and computer scores, as well as the number of draws.

Clear win/loss declaration: Announces the winner at the end of the final round.

When you start the game, you will be prompted to enter the number of rounds you wish to play.

In each round, you will be asked to enter your choice:

Type 'S' or 's' for Snake.

Type 'W' or 'w' for Water.

Type 'G' or 'g' for Gun.

The computer will randomly generate its choice.

The round winner is determined, and the scores are updated.

After the specified number of rounds, the final score will be displayed, and the overall winner will be declared!

# Second Project on Guess the Number Game
A classic command-line implementation of the "Guess the Number" game written in Python. Play against the computer to see if you can guess the secret number in the fewest attempts!

Customizable Range: The user can define the minimum and maximum bounds for the secret number.

Difficulty/Hint Mode: Hints are provided after each guess ("Too High" or "Too Low").

Attempt Counter: Tracks the total number of guesses the player makes.

Score Summary: Declares the winner and the number of attempts taken to guess the number.
